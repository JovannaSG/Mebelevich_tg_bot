## 2.1. Пользовательские сценарии (Frontend)
Главное меню:
1.	🛋 Рассчитать стоимость (Квиз → Лид).
2.	📏 Записаться на замер (Календарь → Заявка).
3.	📸 Оценить проект (Отправка фото/чертежей → Оценка менеджером).
4.	📍 Наши работы (Каталог кейсов с фото).
5.	📞 Контакты (Связь с менеджером, кнопка "Поделиться контактом").

Сценарий "Рассчитать стоимость" (Улучшенный Квиз):
1.	Тип мебели: (Кухня, Шкаф, Детская и т.д.) — Кнопки.
2.	Размеры: (Ширина, Высота, Глубина) — Текст/Числа.
3.	Локация: (Город, Район) — Текст или Геолокация.
4.	Бюджет: (Эконом, Стандарт, Премиум) — Кнопки.
5.	Контакты: (Имя, Телефон) — Текст + Кнопка "Поделиться контактом".
6.	Файлы: (Эскизы, фото) — Опционально.
7.	✅ Проверка данных: Показать резюме заявки. Кнопки: "Подтвердить"/"Редактировать".

Сценарий "Записаться на замер":
1.	Выбор даты (календарь или список дней).
2.	Выбор временного слота.
3.	Подтверждение контакта.
4.	Комментарий к заказу.

## 2.2. Административная часть (Backend & Admin)
Уведомления: Мгновенное сообщение менеджеру в ЛС при новом лиде/записи.

Команды:
*	/leads — Выгрузка последних лидов.
*	/appointments — Список замеров на сегодня/завтра.
*	/stats — Общая статистика (всего пользователей, конверсия в лид).

Рассылка: Возможность отправить сообщение всем пользователям (акции, новости).
